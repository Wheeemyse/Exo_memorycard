Memory

Bienvenue dans le jeu du Memory. Le but est de collecter les paires correspondantes.
il y a 10 paires au total à trouver. Cliquez sur les images pour retourner les cartes.
Lorsqu'une paire est trouvées, les cartes restent face visible. le temps est chronométré
et le nombre de coups également. (un coup equivaut à deux cartes retournées).
Lorsque toutes les cartes sont face visible, c'est gagné!!!

Pour commencer le jeu, appuyez sur le bouton Start!!!
